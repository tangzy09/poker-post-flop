Thin Value

Lesson 22 · Marginal hands that still want a bet · 24 practice questions

Thin value defined

Thin value = betting a marginal hand that beats >50% of villain's calling range. Size down to get called by worse.

When to bet thin

Bet thin when villain's range is capped (checked) or wide (calling station). Avoid thin value vs tight players.

Check medium hands

When unsure if you're ahead, checking is often better than a thin bet that only gets called by better.

Key takeaways

• Thin value targets worse hands in villain's range — size small.
• Bet thin vs capped/wide ranges; check vs tight ranges.
• If only better calls, you're not value betting — check.

FAQ

What is a thin value bet?

A thin value bet is betting a marginal made hand that is only slightly ahead of the hands that call. Sizing is small so worse hands still pay you off.

How do you know if a hand is good enough to value bet thin?

Ask whether worse hands will call more often than better hands raise. If your bet is called mostly by hands you beat, the thin value bet profits.
