# Postflop Coach 翻后训练营 — 产品说明书 V1.0

> 本文定义产品定位、商业化模型与红线。架构与开发约定见 [CLAUDE.md](CLAUDE.md);题库规范见 [docs/CONTENT.md](docs/CONTENT.md)。姐妹产品:[pokerPreFlop](../pokerPreFlop/PRODUCT.md)(翻前训练营)——两站商业化战略保持一致。

## 1. 一句话定位

> 一个**诚实、双语、免费上手**的单挑德州扑克**翻牌后**决策训练器——30 课 699 道经独立盲解验证的实战题,每题带计算式讲解(outs/胜率/赔率/MDF),配间隔重复与每日训练。与「翻前训练营」组成完整的翻前+翻后训练体系。

护城河不是"题最多",而是:**题库不能错**(全量对抗性盲解验证)+ **计算式反馈**(对错都讲数学,绝不编造)+ **中文市场空白**(实测无可玩的中文网页版竞品)。

## 2. 商业化模型(2026-07-03 拍板,两站一致)

### 渠道战略

- **web 现为测试阶段,全免费**——**不是永久承诺**。目的:攒流量、口碑、SEO 收录(58 个双语课程页已收录中)、分享卡传播。
- **正式上线后,付费只发生在 App**(Play/App Store 内购,RevenueCat,沿用 preflop 已建基建)。
- **web 转为试用+导流层**:点到收费内容时**不在 web 解锁**,弹「去 App 下载」引导层(Pro 徽章 + 商店按钮)。
- **先决条件:App 必须先上架**,web 才能开闸锁内容(跳转要有目标)。执行顺序:Capacitor 打包 → Play 上架(复用 preflop 的商店素材流水线)→ 定义 web 锁定范围 → 两站同步开闸。
- **为什么这个模型**:中文用户 web 支付需要国内主体/备案(悉尼 EC2 无备案),App 内购天然绕开;web 免费层保住 SEO 承诺与传播。

### 定价与捆绑

- 定价跟随 preflop,**以 pokerPreFlop/js/purchases.js 头注释为唯一事实源**(2026-07 现价:年 $29.99 主推 / 月 $4.99),RevenueCat 统一。
- **倾向捆绑**:一个 Pro entitlement 同时解锁翻前+翻后两个 App,**不加价**——本站单独可锁的 power features 较薄(见下),捆绑才有价值感;对比 GTO Wizard $480+/年,定位「普通玩家的第一个训练器」。

**切分线已定(2026-07-09,App 端已实装,commit `b3f9b61`):C1–C12 免费,C13–C30 Pro**(取两套内容系统的自然边界;上线后可按留存数据再调)。App 内锁定课卡可点、弹付费墙;web 测试期经 `isProUnlocked()` 恒解锁,不受影响。

⚠ **年龄分级红线(苹果 2026-07 新政)**:个人开发者账号禁发「模拟赌博」app——本产品分级必须保持 `gamblingSimulated=NONE`,对外表述始终是「教学/训练工具,无真钱无下注」;任何新功能不得引入筹码余额、下注输赢等赌博机制,否则 App Store 直接除名风险。

## 3. 红线(任何功能/文案改动都必须遵守)

1. **题库不能错**:每道题的答案必须唯一无争议,经机械质检 + 对抗性盲解验证后才发布(流程见 `.claude/skills/gen-poker-drills`)。
2. **计算式反馈不编数字**:explain.js 只印可计算的 outs/胜率/赔率/MDF;靠隐含赔率定夺的局面只给措辞不印不等式;成对+听牌如实标注「outs 仅计听牌」。
3. **双语**:所有面向用户字符串走 `reg()`/`{en,zh}`,绝不硬编码单语。
4. **商业化文案诚实**:测试期对外说「免费」为真;**正式上线时必须同步调整全部「免费」表述**——title、meta description、58 个课程页 CTA、分享卡、README——不留旧承诺打脸。上线前跑一遍:`grep -rn "免费\|free\|Free" index.html tools/gen-seo-*.js js/i18n.js` 逐条核对。
5. **web 端永不出现支付**:web 的收费触点只做下载引导,不接任何支付渠道。

## 4. 现状(2026-07)

- **内容**:30 课(C1=摸底)699 题,全量盲解验证;计算式反馈引擎。
- **学习系统**:SRS 复习堆(Leitner 1/3/7 天)、每日训练+打卡、连击计分+S/A/B/C 评级、漏洞热图、趋势图、答错反馈闭环(对比条/归因/一键专练)、新手引导。
- **增长**:SEO 全量上线(关键词 head、静态内容块、58 双语课程页、sitemap,GSC 已验证提交);每日成绩分享卡;与翻前站互链闭环。
- **质量**:73 项测试(含 explain 数学锁、SEO 生成物同步锁)+ 699 题全量审计;全局 code review 四批闭环(24 实锤全修)。
- **未做**:App 打包上架(商业化先决条件)、web 锁定交互、支付(按本文模型,永不做 web 支付)。
